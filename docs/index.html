Introduction

Mentor-Mate is an AI-powered educational platform tailored for Sri Lankan Advanced Level (A/L) students. It provides personalized support by answering study-related questions with curriculum-aligned accuracy and offering detailed feedback on model question answers. Using a robust Retrieval-Augmented Generation (RAG) pipeline, Mentor-Mate ensures reliable, curriculum-specific responses while minimizing AI hallucinations. This innovative platform transforms traditional study methods into an interactive and engaging learning experience.

High Level Architecture


The block diagram illustrates the architecture of Mentor-Mate's AI-powered personalized tutor system. When a user asks a question, the platform processes it using a combination of a large language model (LLM), a vector database for semantic search, and a custom retrieval-augmented generation (RAG) pipeline. Personal information, previous chat history, and relevant resources are utilized to provide accurate, curriculum-specific, and context-aware responses. The system integrates chat memory to maintain conversation continuity and leverages embedding-based document retrieval for precise and reliable answers, creating an engaging and effective learning experience for A/L students.


The MentorMate platform features a dedicated Unit-Wise Question Platform designed to help students evaluate their understanding of individual topics in the Biology syllabus. This functionality enables students to engage with model questions structured according to specific syllabus units and receive personalized feedback and grading based on their responses.
